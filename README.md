# peaks
> Get a random Swiss peak, or all of them

# Installation
```Bash
npm install -g peaks
```

# Usage
```Bash
$ peaks --help

  Gets a random Swiss mountain peak or lists all peaks

  Examples
    $ peaks
    Madrisa

    $ peaks --all
    Madrisa
    Rhinerhorn
    Jakobshorn
    ...

  Options
    --all  Get all peak names instead of a random name
```
